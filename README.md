# My Restaurant Website

A responsive restaurant website built using **React** and **Tailwind CSS** with a table booking interface.

## Live Demo

#LINK: https://satya1405.github.io/my-restaurant

##  Features

* Responsive restaurant layout
* Table booking form UI
* Built with reusable React components
* Styled using Tailwind CSS
* Clean and modern design

##  Tech Stack

* React
* Tailwind CSS
* JavaScript
* Vite

##  Sections Included

* Hero Section
* About Us
* Menu Items
* Chef Section
* Table Booking
* Footer
